## 1. 产品概述

中华美食地图是一款参考 TasteAtlas 的交互式美食地图应用，以可缩放的世界地图为底图，重点呈现中国范围内的美食文化。用户可在地图上浏览四大菜系、八大菜系以及 34 个省市自治区的代表性美食，每道美食配有图片、分类、口味、发源地、详细介绍等信息，并支持按"传统美食"与"流行美食"两大类分别筛选。

- 目标用户：美食爱好者、文化研究者、旅行者、对中国饮食文化感兴趣的大众用户
- 解决问题：将分散的中国美食文化信息以地图可视化形式集中呈现，便于探索与学习
- 价值：成为探索中华美食文化的可视化入口，兼具知识性、文化性与审美体验

## 2. 核心功能

### 2.1 用户角色

本应用为纯前端展示型应用，无需用户角色区分，所有访客均可使用全部功能。

### 2.2 功能模块

1. **地图主页**：世界地图底图、美食标记点、缩放平移、分类筛选、菜系快速跳转
2. **美食详情面板**：点击标记后展示美食完整信息（图片、名称、分类、口味、发源地、详细介绍）
3. **筛选与图例**：传统美食 / 流行美食切换、菜系筛选、分类筛选、口味筛选
4. **侧边索引**：按菜系、按省份浏览美食列表，支持搜索

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 地图主页 | 地图容器 | Leaflet 地图，中文底图，支持任意缩放与平移，默认聚焦中国范围 |
| 地图主页 | 美食标记 | 根据缩放级别聚合/展开标记，标记样式区分传统/流行两类 |
| 地图主页 | 顶部导航栏 | Logo、搜索框、菜系快速跳转、筛选切换 |
| 地图主页 | 左侧索引面板 | 按菜系/省份分组的美食列表，可搜索可点击定位 |
| 地图主页 | 右侧详情面板 | 点击标记后展示美食详情卡片 |
| 地图主页 | 底部图例 | 标记含义说明、筛选状态显示 |

## 3. 核心流程

### 3.1 主要用户流程

用户进入应用 → 看到聚焦中国的地图与美食标记 → 通过缩放/平移探索不同区域 → 点击标记查看详情 → 使用筛选切换传统/流行美食 → 通过左侧索引按菜系或省份浏览 → 点击列表项定位到地图对应位置

## 4. 界面设计

### 4.1 设计风格

**美学方向：东方水墨 × 现代编辑设计**

将传统中国水墨画的意境与现代美食杂志的编辑设计语言融合，营造既有文化厚度又有当代审美的美食地图体验。

- **主色调**：
  - 背景：宣纸米白 `#F5EFE0` / 墨黑 `#1A1614`
  - 主色：朱砂红 `#C8392E`（传统中国红，用于强调与传统美食标记）
  - 辅色：靛青 `#2C5F7C`（用于流行美食标记与链接）
  - 点缀：赭石 `#A0623A`、金箔 `#C9A961`
- **字体**：
  - 中文标题：思源宋体 / Noto Serif SC（庄重典雅）
  - 中文正文：思源黑体 / Noto Sans SC（清晰易读）
  - 数字与拉丁：Cormorant Garamond（编辑感衬线体）
- **按钮风格**：圆角矩形，朱砂红主按钮带细微阴影，次级按钮为描边样式
- **布局**：左侧索引面板（可收起）+ 中央地图 + 右侧详情面板（按需弹出）+ 顶部导航
- **图标**：使用 lucide-react 线性图标，搭配自定义美食分类图标
- **纹理**：地图容器外区域加入宣纸纹理与水墨晕染装饰元素

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 地图主页 | 顶部导航栏 | 米白底，朱砂红 Logo，思源宋体标题，搜索框带毛笔笔触装饰，菜系标签横向滚动 |
| 地图主页 | 左侧索引面板 | 半透明米白卡片，分组折叠列表，每项含缩略图+名称+地区标签，hover 朱砂红高亮 |
| 地图主页 | 地图容器 | 占据中央主区域，圆角边框带水墨晕染阴影，缩放控件置于右下 |
| 地图主页 | 美食标记 | 传统美食：朱砂红圆形带印章感图标；流行美食：靛青水滴形标记 |
| 地图主页 | 右侧详情面板 | 米白卡片从右侧滑入，顶部大图，下方信息分区，发源地标注"传/据说" |
| 地图主页 | 底部图例 | 固定底部半透明条，标记说明 + 当前筛选状态标签 |

### 4.3 响应式

- 桌面优先（1280px+）：三栏布局完整呈现
- 平板（768-1280px）：左侧索引可收起为图标条，详情面板覆盖式弹出
- 移动端（<768px）：地图全屏，索引与详情改为底部抽屉式，筛选改为顶部下拉

### 4.4 3D 场景

本应用不涉及 3D 场景。
